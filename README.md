# 🎫 Melon Ticket 좌석 프리미엄 가격 시각화 (show-resell)

> **NCT WISH 2ND ANNIVERSARY FANMEETING** 좌석별 리셀 프리미엄 가격을 시각화하는 순수 프론트엔드 UI 프로젝트

🔗 **Live Demo**: `https://mmilly-wish.github.io/show-resell/`

---

## 📌 프로젝트 개요

멜론 티켓 좌석 선택 화면을 UI 수준에서 재현하고, 한국(KR) 및 중국(CN) 플랫폼에서 수집된 좌석별 **리셀 프리미엄 가격**을 밀도 기반 히트맵으로 시각화합니다.

> ⚠️ 이 프로젝트는 **예매 기능 없음** — UI 시각화 및 데이터 분석 전용입니다.

---

## 📊 데이터 수집 및 주의사항 (Disclaimer)

본 프로젝트에 사용된 데이터는 플미 티켓 근절과 정보 공유를 목적으로 수집되었으나, 기술적 한계로 인한 오류가 있을 수 있습니다.

- **수집 데이터**: 한국(KR) 및 중국(CN) 리셀 플랫폼 프리미엄 티켓 매물 통합 (약 500건)
- **수집 기준일**: **2026.02.21**
- **수집 방법**: 이미지 기반 **OCR(광학 문자 인식)** 기술 활용 추출
- **⚠️ 주의사항**: 
    - 모든 데이터는 2026.02.21 특정 시점에 수집된 정보로, 실시간 매물 현황과 차이가 있을 수 있습니다.
    - **OCR 기술의 특성상** 문자 오인식으로 인해 구역, 열, 가격 등 상세 정보에 일부 오류가 존재할 가능성이 있습니다.
    - 본 사이트의 정보는 참고용일 뿐이며, 실제 거래 데이터와 100% 일치하지 않을 수 있음을 유의하시기 바랍니다.

## 🎨 기능

| 기능 | 설명 |
|------|------|
| 좌석 그리드 | 잠실실내체육관 각 구역 및 열별 좌석 배치 재현 |
| 밀도 히트맵 | 해당 열의 매물 개수(밀도)에 따라 `rgb(244, 67, 54)` 농도 차등 적용 |
| 툴팁 | hover 시 해당 열의 총 매물 수 및 상세 가격 정보 표시 |
| 미니맵 | 구역별 원형 미니맵을 통한 직관적인 구역 전환 |
